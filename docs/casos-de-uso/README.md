# Casos de uso de Fulgor Medieval

Cada caso de uso describe una acción que un actor (jugador, sistema) puede
hacer en el juego, con su flujo normal y sus casos de error. Se numeran
CU-01, CU-02, ... en el orden en que los construimos.

## Índice

| ID | Nombre | Hito | Estado |
|---|---|---|---|
| CU-01 | Pasar turno | 1 - Núcleo | Definido |

## Plantilla

```markdown
# CU-XX — Nombre del caso de uso

- **Actor:** quién lo ejecuta (jugador, sistema...)
- **Precondiciones:** qué tiene que ser verdad antes de empezar
- **Flujo principal:** pasos numerados del camino feliz
- **Flujos alternos:** qué pasa si algo falla o el jugador hace otra cosa
- **Postcondiciones:** qué es verdad al terminar
```
